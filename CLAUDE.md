# Project Instructions

## Git
- Never add "Co-Authored-By" lines to commit messages. All commits should be authored solely by the user.

## Project Overview
End's Dawn is a UE5 narrative strategy RPG based on the novel End's Dawn.
The game begins after the Demon King's defeat. The king is comatose, the crown
prince is dead, Leonhardt is missing, and the remaining princes begin a struggle
for succession.

## Playable Routes
- Maximilian, Alaric, Wilhelm, Conrad, Rupert, Gerald, Casimir
- Leonhardt is a special unlockable route.

## Design Priorities
1. Canon fidelity
2. Shared scalable systems
3. Data-driven content
4. Branching consequences
5. Modular UE5 C++ architecture

## Non Goals
- Multiplayer
- Live service systems
- Procedural world generation
